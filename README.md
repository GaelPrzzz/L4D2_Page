Página web de Left 4 Dead 2 <br>
  Esta es una página web informativa sobre uno de mis videojuegos favoritos y un proyecto personal que hice mientras aprendía desarrollo web. Es uno de mis primeros proyectos, donde estuve probando y practicando lo básico de HTML, CSS y un poco de JavaScript.

Sobre el proyecto <br>
  La idea fue crear una página sencilla para practicar cómo estructurar un sitio web, aplicar estilos y agregar algo de interactividad. También quise hacer algo relacionado con un videojuego que marcó mi infancia, así que el proyecto tiene un toque personal y algo de nostalgia.

Tecnologías usadas
- HTML
- CSS
- JavaScript

Objetivo <br>
  Más que hacer algo perfecto, el objetivo fue aprender, experimentar y entender mejor cómo funciona el desarrollo web desde cero.

Estado <br>
  Proyecto simple y funcional. Probablemente lo siga mejorando conforme aprenda más cosas.

Autor <br>
  Proyecto hecho como práctica personal mientras aprendo desarrollo web.
